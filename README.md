## Usage
### maven
```
<dependency>
    <groupId>com.lvonce</groupId>
    <artifactId>athena</artifactId>
    <version>1.0</version>
</dependency>
```
### gradle
```
compile group: 'com.lvonce', name: 'athena', version: '1.0.0'
```
